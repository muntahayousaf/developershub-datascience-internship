# 🌸 Task 1: Exploring and Visualizing the Iris Dataset

## 📌 Objective
The goal of this task is to understand how to load, inspect, summarize, and visualize a dataset using Python. The Iris dataset is one of the most popular datasets for learning data analysis and visualization techniques.

---

## 📦 Dataset
**Iris Dataset** (loaded directly using Seaborn)

The dataset contains **150 flower samples** from **three iris species**:

- Setosa
- Versicolor
- Virginica

Each sample includes the following features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

---

## 🔧 Approach

### 1. Data Loading
- Loaded the dataset using Seaborn.
- Imported required libraries such as Pandas, Matplotlib, and Seaborn.

### 2. Data Exploration
- Checked dataset shape.
- Inspected column names and data types.
- Viewed sample records.
- Checked for missing values.
- Checked for duplicate records.

### 3. Data Visualization
Created multiple visualizations to better understand the dataset:

- Scatter Plot
- Histogram
- Box Plot
- Pair Plot

---

## 📊 Visualizations

| Visualization | Purpose |
|--------------|---------|
| Scatter Plot | Shows relationships between sepal and petal measurements |
| Histogram | Displays feature distributions across species |
| Box Plot | Identifies spread and potential outliers |
| Pair Plot | Visualizes relationships between all features simultaneously |

---

## 💡 Key Insights

### 🌱 Species Separation
- **Setosa** is clearly distinct from the other two species.
- It can be easily separated based on petal measurements.

### 📏 Important Features
- **Petal Length** and **Petal Width** are the most useful features for distinguishing species.
- These features show clear clustering among flower types.

### 📦 Outliers
- **Sepal Width** contains a few outliers.
- No extreme anomalies were observed.

### ⚖️ Balanced Dataset
- The dataset is perfectly balanced.
- Each species contains **50 samples**.
